发布于：2020-01-26 19:00   标签分类： ios swift

语法跟javascript类似。每句话后面不需要分号（跟python类似）。

声明变量并赋值

var str = "hello"

这句话等于

var str : String
str = "asdf"

或

var str : String = "asdf"

跟javascript不同的是变量一旦确定了类型就不能变了。下面这样会报错。

var count = 1
count = "aaaa"

声明常量用let

let pi = 3.1415926

print()函数可以打印输出内容

print("aaa")

预定义类型

Int、UInt（无符号整数）、Float、Double 4种基本数值类型，swift不会做默认的类型转换 下面的语句会报错

let a = 123
let b = 3.1
let c = a+b

可以强制类型转换

let a = 123
let b = 3.1
let c = Double(a)+b

不支持++和–

class CircleClass1 {
    var radius: Double = 1

    init() {

    }

    init(radius: Double) {
        self.radius = radius
    }

    var area: Double {
        return M_PI * radius * radius
    }

    func circumference: Double {
        get {
            return 2 * M_PI * radius
        }
        set (value) {
            radius = value / (2*M_PI)
        }
    }

    var description: String {
        return "Circle of radius \(radius)"
    }
}

// 协议=抽象类？
protocol Resizable {
    var width: Float {get set}
    var height: Float {get set}

    init(width: Float, height: Float)
    func resizeBy(wFactor: Float, hFactor: Float) -> Void
}

class Rectangle : Resizable {
    required init(width: Float, height: Float) {
        self.width = width
        self.height = height
    }
    func resizeBy(wFactor: Float, hFactor: Float) -> Void {
        width *= wFactor
        height *- hFactor
    }
}
