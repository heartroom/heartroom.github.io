php生成图片问题

发布于：2020-01-20 16:10   标签分类： php

php生成图片，在html的img标签中可以正常显示，直接访问网址乱码的问题解决：

今天做php生成图片的时候，出现了一个问题。代码实际上挺简单的，如下：

header("Content-type:image/png");
$a = imagecreate(100,100);
imagecolorallocate($a,255,0,255);
imagepng($a);
imagedestroy($a);

当我新建一个html页面，使用img标签是图片可以正常显示，网上找了很久，试了很多方法，终于找到了解决方法

解决方法： 修改上门的代码为

ob_end_clean();//添加这句就可以了
header("Content-type:image/png");
$a = imagecreate(100,100);
imagecolorallocate($a,255,0,255);
imagepng($a);

这样图片就可以直接通过网址访问了
