ubuntu虚拟机扩容

发布于：2020-01-29 09:00   标签分类： linux virtualbox

今天发现虚拟机的容量为0了，没办法之前给的容量太小了，所以需要扩容。

网上找了很多文章，这篇最靠谱

Ubuntu14.0使用gparted调整分区大小

基本按照这个文章写的顺序就可以了，但是其中的gparted版本太旧了，在ubuntu18.04.1上安装，会提示安装失败，e2fsck版本太低。

需要到https://gparted.org/download.php 这里下载最新的gparted版本。

具体步骤和命令在这里简单记录一下

// 在windows命令行里输入下面的命令，先在windows端扩容
cd C:\Program Files\Oracle\VirtualBox 

VBoxManage list hdds

// 0bd9c696-1735-48ce-81cf-04e9f64c2418 和 51200 根据自己机器和要扩容的大小 修改
VBoxManage modifyhd 0bd9c696-1735-48ce-81cf-04e9f64c2418 --resize 51200

下载最新的gparted

https://sourceforge.net/projects/gparted/files/gparted-live-stable/1.1.0-1/gparted-live-1.1.0-1-i686.iso/download?use_mirror=jaist

在虚拟机里选择ubuntu，点击“设置”，存储介质里，添加新的虚拟光驱（把下载的gparted ios文件加进来）。

然后启动虚拟机

在启动页面 按 esc。进入gparted安装页面

一路按enter就行。

然后正常会出现磁盘列表，在主分区上点击右键，选择resize。然后调整到最大，点击apply，然后等扩容完成后，点击Exit，reboot就行了。
