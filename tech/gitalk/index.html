通过gitalk为博客添加评论功能

发布于：2020-01-30 06:00   标签分类： jekyll

主要参考了这两篇文章：

Mac 一步一步教你在Jekyll博客添加评论系统

在个人博客里添加评论系统–Gitalk

一、申请一个Github OAuth Application

二、在jekyll博客添加gitalk

在文章页模板(post.html)中，添加如下代码。

<div id="gitalk-container"></div>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/gitalk@1/dist/gitalk.css">
<script src="https://cdn.jsdelivr.net/npm/gitalk@1/dist/gitalk.min.js"></script>
<script>
var gitalk = new Gitalk({
    id: '2020-01-30 06:00:00 -0600',
    clientID: '29e59d091901a686d79e',
    clientSecret: 'f771ab0b454e0fbd1f8356d3f400e571172118f3',
    repo: 'heartroom.github.io',
    owner: 'heartroom',
    admin: ['heartroom'], 
	labels: ['Gitalk'],
})
gitalk.render('gitalk-container')
</script>

三、添加鉴权代码

在_config.yml里，添加下面的代码，clientID，clientSecret是第一步产生的

gitalk:
    enable: true
    owner: heartroom
    repo: heartroom.github.io
    clientID: ****c08c2ec52afbcb30
    clientSecret: *******79e7b42cf782a1d2d1b5a410d27c5ab57
    admin: heartroom
