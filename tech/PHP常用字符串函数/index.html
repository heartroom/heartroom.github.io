Php常用字符串函数

发布于：2019-12-27 00:00   标签分类： php

  1. strlen() 获取字符串长度

    strlen()、mb_strlen()，不同之处在于mb_strlen()第二个参数可以用于指定字符编码。

    PHP内置的字符串长度函数strlen无法正确处理中文字符串，所以它得到的只是字符串所占的字节数。对于UTF-8编码的中文，一个汉字占三个字节。

     <?php
 　　$a = '你好a';
 　　echo strlen($a);//输出的结果是7
 ?>
    
 我知道有时候你想要的结果是3。那么怎么解决呢？就是用mb_strlen()。例如：
 <?php
 　　$a = '你好a';
 　　echo mb_strlen($a,utf-8);//输出是3
 ?>
 对于mb_strlen()如果不写第二个参数，就会使用内部编码，效果就和strlen()一样。

  2. strpos() 在字符串内查找一个字符或字符串第一次出现的位置

  3. trim() 函数 是使用来清除字符串前后的空格字符或其它预定义字符

  4. htmlspecialchars() 将< >这些符号转为<等等，htmlspecialchars_decode这个是htmlspecialchars函数的反函数

  5. str_replace() 函数

    把字符串 “Hello world!” 中的字符 “world” 替换为 “Shanghai”：

     <?php
 echo str_replace("world","Shanghai","Hello world!");
 ?>

  6. substr() 从字符串中获取其中的一部分

    <?php
 echo substr('abcdef', 1);      //输出 bcdef
 echo substr('abcdef', 1, 2);   //输出 bc
 echo substr('abcdef', -3, 2);  //输出 de
 echo substr('abcdef', 1, -2);  //输出 bcd
 ?>
